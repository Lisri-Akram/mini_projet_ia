Heart Disease Patient ClusteringThis project applies Unsupervised Machine Learning to a heart disease dataset. Installation & RequirementsTo run this script, you will need Python 3.x and the following libraries:Bashpip install numpy pandas matplotlib seaborn scikit-learn
 UsageEnsure heart.csv is in the same directory as the script.Run the script:Bashpython heart_clustering.py
 Evaluation MetricsThe model is evaluated using three key metrics:Inertia: Measures how tightly the clusters are packed.Silhouette Score: Measures how similar an object is to its own cluster compared to other clusters (ranges from -1 to 1).Davies-Bouldin Index: The ratio of within-cluster distances to between-cluster distances (lower is better).
